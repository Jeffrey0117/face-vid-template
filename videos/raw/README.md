# 影片素材資料夾

這個資料夾用於放置待處理的面相影片檔案。

## 使用說明

1. **放置影片**：將您的面相影片檔案放入此資料夾
2. **檔案格式**：支援 `.mp4`, `.avi`, `.mov`, `.mkv`, `.wmv`, `.flv` 等格式
3. **檔案命名**：建議使用面相類型命名，例如：
   - `典型妾相.mp4`
   - `頂級貴相.mp4`
   - `克夫相.mp4`
   等等

## 注意事項

- 檔案名稱建議不超過 4 個字，避免影片標題過長
- 系統會自動掃描此資料夾中的所有影片檔案
- 處理完成後，對應的剪映專案會以 `面相專案_檔案名稱` 的格式命名

## 範例結構

```
videos/raw/
├── 典型妾相.mp4
├── 頂級貴相.mp4
├── 克夫相.mp4
└── 其他面相影片...
```

⚠️ **重要**：此資料夾中的檔案不會被提交到 Git 倉庫，只是作為本地處理使用。
